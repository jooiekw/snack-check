# ✅ Snack Check

### 🇰🇷 내가 먹는 간식, 성분 알고 먹기

**Snack Check**는 제품의 크기에 따라 **실제 섭취량(1봉지)** 또는 **100g 기준**으로 최적화된 영양성분을 제공하는 스마트 간식 성분 확인 서비스입니다.
복잡한 포장지 뒷면 대신, 직관적인 데이터로 더 건강한 간식 생활을 돕습니다.

## 📌 주요 기능
* **스마트 영양 기준:** 100g 이하 소포장 제품은 **총 내용량(1봉지)** 기준으로, 대용량 제품은 **100g 당** 기준으로 자동 변환하여 보여줍니다.
* **다양한 카테고리:** 과자뿐만 아니라 라면, 초콜릿, 캔디 등의 정보를 제공합니다.
* **알레르기 필터:** 우유, 대두, 밀 등 특정 알레르기 성분이 포함된 제품을 제외하고 볼 수 있습니다.
* **맞춤 정렬:** 칼로리 낮은 순, 단백질 높은 순, 당류 낮은 순 등으로 제품을 정렬해 찾을 수 있습니다.

## 🛠 사용 기술
* **Python & Streamlit:** 웹 애플리케이션 구현
* **Pandas:** 대용량 데이터 전처리 및 분석
* **SQLite:** 경량 데이터베이스 구축
* **GitHub:** 버전 관리 및 배포

### 🇺🇸 Know What You Snack

**Snack Check** is a smart snack information service that provides optimized nutrition facts based on either **actual serving size (1 package)** or **per 100g**, depending on the product size.
We help you make healthier choices with intuitive data, without the hassle of studying complex package labels.

## 📌 Key Features
* **Smart Serving Standard:** Automatically displays nutrition facts based on **Total Content (1 package)** for small items (<100g), and **per 100g** for bulk items.
* **Various Categories:** Provides information on snacks, ramen, chocolate, candy, and more.
* **Allergy Filter:** Excludes products containing specific allergens like milk, soy, wheat, etc.
* **Custom Sorting:** Sort products by Low Calories, High Protein, Low Sugar, and more.

## 🛠 Tech Stack
* **Python & Streamlit:** Web application development
* **Pandas:** Data preprocessing and analysis
* **SQLite:** Lightweight database construction
* **GitHub:** Version control and deployment

## 📊 Data Source
* Ministry of Food and Drug Safety (MFDS): Integrated Food Nutrition Database
